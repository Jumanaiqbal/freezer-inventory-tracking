# Sanar Freezer - Inventory Tracking System

**Professional inventory management system for commercial kitchens.**

## Quick Start

### Installation
```bash
npm install
npm start
```

### Database Setup (REQUIRED)
Before first use, follow [DATABASE_SETUP.md](DATABASE_SETUP.md) to:
- Update database schema
- Create history table
- Enable real-time
- Seed initial data

```bash
node seed.js
```

## Features

✅ **Live Dashboard** - Real-time inventory with color-coded stock levels  
✅ **Stock Management** - Add/remove items with worker tracking  
✅ **Search** - Find items instantly  
✅ **Audit Trail** - Track all changes in history table  
✅ **Responsive** - Works on mobile, tablet, desktop  
✅ **Real-time Sync** - Instant updates across all users
